An Analytic Discrete Fracture Network (AnDFN)
=============================================

AnDFN is a computer program that calculated the flow in a discrete fracture network (DFN) using the Analytic Element
Method (AEM).

More information can be found in the `about <about>`_ tab.

.. image:: example.svg
   :width: 800
   :alt: example

Installation
------------
The package is uploaded to the Python Package Index (PyPI) and can be installed using pip.

Install the package:

.. code-block:: console

    pip install andfn

Update the package to the latest version:

.. code-block:: console

    pip install andfn --upgrade

Uninstall the package:

.. code-block:: console

    pip uninstall andfn
